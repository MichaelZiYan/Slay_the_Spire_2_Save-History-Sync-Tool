# 🃏 Slay the Spire 2 - Save & History Ultimate Sync Tool

### 杀戮尖塔2 存档与战绩全能同步工具 (隐私保护版)

本工具专为《杀戮尖塔2》玩家打造，旨在完美解决 Steam 原版、Mod 环境、GSE 补丁版以及不同 Steam 账号之间存档不互通、战绩记录（History）丢失的痛点。

## ✨ 核心特性

* **🔍 隐私安全自动探测**：采用动态探测算法，通过系统环境变量与注册表自动寻找所有可能的存档路径，**代码不含任何个人 SteamID 或用户名**，保护隐私的同时实现全自动识别。
* **💾 智能路径记忆**：首次运行自动扫描，后续操作自动记忆。支持手动增删路径并永久缓存于本地 `paths_config.txt`。
* **📊 战绩并集同步 (History Merge)**：不仅仅同步进度！一键合并所有版本的 `history` 文件夹，自动去重补全 `.run` 文件，让你的对战统计在所有版本下保持完整。
* **🏆 安全解锁全成就**：提供一键覆盖全成就存档功能，并在操作前**强制备份**当前最新进度至 `我的原始存档备份.save`，确保万无一失。
* **🚀 极致启动体验**：提供 `.exe` 直行版本及无窗口运行的 `.pyw` 源代码，告别黑窗口。

## 📂 仓库文件指南

| 文件名 | 说明 |
| --- | --- |
| **`StS2_Sync_Tool.exe`** | **推荐使用**。打包好的独立程序，双击即用，无需安装 Python。 |
| **`启动.bat`** | 批处理启动脚本，适合有 Python 环境的用户，出错时可查看报错。 |

| **`paths_config.txt`** | (运行后生成) 本地缓存文件，记录你自定义或探测到的路径。 |

## 🚀 快速开始

### 1. 准备工作

* 如果需要解锁全成就，请确保仓库中的 **`全存档.save`** 已下载并与工具放在同一文件夹内。

### 2. 运行工具

* **普通用户**：直接运行 `StS2_Sync_Tool.exe`。
* **开发者/Python用户**：运行 `启动.bat` 或 `python StS2_Ultimate_Sync.pyw`。

### 3. 操作流程

1. **确认路径**：启动后，工具会自动填入探测到的路径。如需增加，请手动在文本框按行输入。
2. **同步进度**：点击“同步最新进度”，工具将以最后修改时间最晚的文件为基准同步所有版本。
3. **同步战绩**：点击“同步对战记录”，合并所有版本的历史战斗图表。

## ⚠️ 注意事项

1. **关闭游戏**：在执行任何同步或覆盖操作前，请确保《杀戮尖塔2》已完全退出，否则存档可能被锁定。
2. **隐私声明**：本工具不收集、不上传任何个人数据。自动探测逻辑仅在本地运行，产生的缓存文件建议不要上传至公共仓库。
3. **版本声明**：本工具针对《杀戮尖塔2》抢先体验版开发，将随游戏更新持续维护。
